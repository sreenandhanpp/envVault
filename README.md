# EnvVault

EnvVault is a MERN-style environment variable manager prototype. This implementation focuses on a polished React frontend with dummy data so the product flow can be reviewed without a backend connection.

## Features shown

- Project cards for multiple apps and services.
- Environment-specific variables for Development, Staging, and Production.
- Secure masked values, copy/edit/delete actions, and search UI.
- Team and permission management for Owner, Admin, Member, and View Only roles.
- Activity feed showing recent variable updates and exports.
- Export panel for downloading `.env` files.

## Getting started

```bash
npm install
npm run dev
```

Build for production:

```bash
npm run build
```
